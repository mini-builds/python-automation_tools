Bulk File Renamer (Python)

This Python script renames multiple files in a folder using a custom prefix and automatic numbering.

Features
- Batch renaming
- Prefix + numbering
- Preview before execution
- Skips folders automatically

Example

Before:
IMG_123.jpg
IMG_124.jpg

After:
Vacation_1.jpg
Vacation_2.jpg

Usage

1 Run the script
2 Enter folder path
3 Enter prefix
4 Confirm renaming
